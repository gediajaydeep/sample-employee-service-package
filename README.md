The `employee_service` package provides the following out-of-the-box features:

* **Automated Payroll Logic**: Built-in logic for calculating Net Salary based on regional tax rates (e.g., India at 10%, USA at 12%).

* **Flexible Filtering**: Advanced querying capabilities to filter employees by name, role, or country.
* **Dynamic Updates**: Smart repository logic that only updates modified fields, preserving existing database integrity.
* **Global Analytics**: Real-time aggregate metrics including average, minimum, and maximum salary statistics.
